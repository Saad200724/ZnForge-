# ZnForge Landing Page

## Overview
A modern landing page for ZnForge - a software development agency. This is a frontend-only React application with Vite, TypeScript, and Tailwind CSS.

## Project Structure
- `src/` - Main source code
  - `components/` - React components
    - `landing/` - Landing page sections (Hero, Services, Portfolio, Pricing, FAQ, etc.)
    - `ui/` - Shadcn UI components
  - `pages/` - Page components (Index, NotFound)
  - `hooks/` - Custom React hooks
  - `lib/` - Utility functions
  - `assets/` - Static assets (images)

## Tech Stack
- React 18 with TypeScript
- Vite for build tooling
- Tailwind CSS for styling
- Shadcn UI components
- React Router DOM for routing
- TanStack React Query

## Running the Project
- Development: `npm run dev` (runs on port 5000)
- Build: `npm run build`
- Preview: `npm run preview`

## Deployment
Configured for static deployment. The build output goes to the `dist` directory.
